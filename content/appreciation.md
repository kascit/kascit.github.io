## Core Site Stack Thanks

The site is built and maintained on top of tools that deserve direct credit:

- Zola for static site generation and templating.
- Tailwind CSS for utility-first styling.
- DaisyUI for component primitives and theme ergonomics.
- Font Awesome for iconography used across UI surfaces.

## Search, Rendering, and Interaction Credits

- Fuse.js and ElasticLunr for fast client-side search experiences.
